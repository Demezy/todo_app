# todo_app

Flutter app to manage your todos.

## Stack

- riverpod
- flutter_hooks
- freezed

## Features
- add tasks
- hide done tasks
- swipes
  - animation
  - open menu via short swipe
  - perform action via long swipe
- clean architecture!
- app icon
